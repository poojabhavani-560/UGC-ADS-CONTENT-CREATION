# AI Content Marketing using UGC Ads

## 📌 Project Overview
This project focuses on creating **high-converting UGC (User Generated Content) ad scripts** using AI.

Modern brands prefer **authentic, relatable content** over traditional polished ads. This system helps generate such content using structured prompts and frameworks.

---

## 🎯 Objective
The goal of this project is to:

- Generate UGC-style ad scripts using AI
- Create multiple hooks for the same product
- Follow proven marketing frameworks (Problem → Solution → CTA)
- Adapt content for platforms like Instagram Reels, YouTube Shorts
- Build a reusable prompt system for real-world usage

---

## 🏷️ Selected Product

**Product Name:** GlowSkin Vitamin C Face Serum  
**Category:** Skincare (D2C Brand)

### Target Audience:
- Age: 18–35
- People facing dull skin, dark spots, uneven tone

### Problem:
- Dull skin
- Acne marks
- Uneven skin tone

### Solution:
- Vitamin C serum for brighter and clearer skin
